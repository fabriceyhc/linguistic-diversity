## Methodology

### Selection Algorithms

1. **Full Dataset**: Train on all available instruction-response pairs (baseline)
2. **Random 10%**: Randomly sample 10% of the data (baseline)
3. **Diversity 10%**: Select 10% using greedy diversity maximization:

   - Embed all samples using `all-MiniLM-L6-v2`
   - Iteratively select samples that maximize minimum distance to selected set
   - For large datasets (>100k), use K-means clustering + centroid selection
